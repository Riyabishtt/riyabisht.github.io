Airline Management System

Description: This project is a comprehensive airline management system designed to streamline various operations involved in airline management. It includes functionalities such as flight scheduling, booking management, passenger check-in, and ticketing.

Technologies Used: Java, Spring Boot, MySQL, Thymeleaf

Features:

  • Flight Schedule Management
  • Booking and Reservation System
  • Passenger Check-in and Ticketing
  • Admin Dashboard for Flight and Booking Management

Share Market Management System

Description: This project is a share market management system aimed at tracking and managing stock trades and investments. It provides real-time stock prices, trade execution, portfolio management, and historical data analysis.

Technologies Used: Python, Django, PostgreSQL, JavaScript

Features:

  • Real-time Stock Price Tracking
  • Trade Execution and Portfolio Management
  • Historical Data and Analytics
  • User Account Management and Reporting
